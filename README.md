# CIS172-PokemonGame-2026
A Pokémon game using Python, for our CIS-172 project. 

How to run:
    1. Make sure Python and Pygame Zero is installed
    2. In the terminal (VSCode or GitBash), use this command: "pgzrun main.py"

How to play: 
    1. Enter your name and press ENTER
    2. Choose your starter Pokemon using the 1, 2, 3 keys 
    3. During the battle: choose a move by using the 1, 2 keys
    4. Battle continues until one Pokemon faints
